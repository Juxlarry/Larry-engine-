## Example (Optional)

```javascript


const universal_validator = require ("universal_validator_pack");

let number = "02003564789"; //Enter a ten 10 digit Ghana number of any network
const phone = number;

const phoneNumberValidator = new universal_validator(phone);

console.log(`Let's check if Phone number ${phone} is valid: ${phoneNumberValidator.validatePhoneNumber()}`);

## Installation

- All the `code` required to get started
- Images of what it should look like

Download package from npm locally into your environment from npm by running line below


### Clone

- Clone this repo to your local machine using `https://github.com/Juxlarry/Universal_Validator.git`

### Setup

> update and install this package first

```shell
$ <terminal> update
$ <npm i universal_validator_pack>
```

> now install npm 

```shell
$ npm install
```
